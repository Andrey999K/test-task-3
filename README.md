# Тестовое задание

## Технологии

- Next.js 16.2.1
- React 19.2.4
- TypeScript 5
- Ant Design 6.3.4
- Recharts 3.8.1
- Tailwind CSS v4

## Установка

```bash
npm install
```

## Запуск

```bash
npm run dev
```

## Структура

```
src/
├── app/          # Страницы
├── components/   # Компоненты
├── hooks/        # Кастомные хуки
├── lib/          # Mock данные
├── types/        # TypeScript типы
└── utils/        # Утилиты
```

## Страницы

- **/** — Дашборд со статистикой и графиками
- **/citizens** — Таблица граждан с фильтрацией

## Данные

Приложение использует моковые данные из `@/lib/mock-data`.
